# Generator-kickstart

A generator for Yeoman.

## Getting started
- Make sure you have [yo](https://github.com/yeoman/yo) installed:
    `npm install -g yo`
- Install the generator: `npm install -g generator-kickstart` (install from github directly!)
- Run: `yo kickstart`
- Run: `grunt install`
